# Shape Recognition Model

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

The source code for shape recognition model, including

- Create, load samples and detect
- Change samples database

## Requirements

- Torch
- Pillow
- numpy

## Usage

You can see the example in the test.ipynb file

Basic usage
```python
from lib import ShapeModel
from PIL import Image
model = ShapeModel(model_path='[path to model file]', samples_root='[path to samples folder]')

image = Image.open('[path to image]').convert('RGB')

model(image)
```

Samples dir structure:
```sh
root:
---img-1
---img-2
---img-3
```
